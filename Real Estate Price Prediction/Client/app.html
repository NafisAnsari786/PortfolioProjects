Banglore Housing Prices Prediction

A web application for predicting housing prices in Bangalore using Machine Learning.

About the Project

This app uses machine learning to predict the price of houses in Bangalore. It loads a pre-trained machine learning model, which takes into account various features of the house, such as the number of rooms, the number of bedrooms, the square footage, and the neighborhood's location. Additionally, the model takes into account factors such as the proximity to major areas, public transport, and other location-based features. The app processes the input data and outputs a predicted house price based on the given attributes.
